# YouTube Clone - React Video Gallery

A responsive YouTube-style video gallery built with React using the FreeAPI YouTube endpoint.  
This project demonstrates API fetching, pagination, loading states, and responsive UI design.

---

## 🚀 Features

- 📺 Fetch YouTube videos dynamically from API
- 🔄 Pagination with smart page grouping
- ⏳ Loading state while fetching videos
- 🎨 Responsive video card layout
- ▶ Direct video links to YouTube
- 👀 Video metadata display:
  - Title
  - Channel name
  - Views
  - Published date
  - Duration

---

## 🛠 Tech Stack

- React
- JavaScript (ES6+)
- CSS
- FreeAPI YouTube API

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/youtube-clone.git
```

2. Navigate to project folder

```bash
cd youtube-clone
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

---

## 🔗 API Used

```bash
https://api.freeapi.app/api/v1/public/youtube/videos
```

Example:

```bash
https://api.freeapi.app/api/v1/public/youtube/videos?page=1&limit=6
```
