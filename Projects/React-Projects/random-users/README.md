# Random User Profile Card

A React-based user profile application that fetches random user data from the Random Users API and displays it in a clean, modern profile card UI.

The project focuses on:
- Working with APIs in React
- Understanding JSON response structures
- Managing state using React Hooks
- Building responsive and reusable UI components

## Features

- Fetch random users from API
- Next and Previous user navigation
- Loading state handling
- Modern dark-themed profile card UI
- Responsive design

## API Used

Random Users API:

https://api.freeapi.app/api/v1/public/randomusers

## Technologies Used

- React
- JavaScript (ES6+)
- CSS3
- Fetch API

## Concepts Practiced

- `useState`
- `useEffect`
- API fetching
- Conditional rendering
- Event handling
- Component-based architecture

## Project Goal

The main goal of this project is to learn how to:
- Read API documentation
- Understand API response structures
- Fetch and display dynamic data
- Create a clean and interactive user interface in React
