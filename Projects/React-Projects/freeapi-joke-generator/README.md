# Random Joke Generator 😂

A modern and interactive React app that fetches random jokes from a public API and lets users browse through jokes with a sleek glassmorphism UI.

---

## 🚀 Features

- 🎭 Fetch random jokes from API
- ⏭ Browse next and previous jokes
- ⚡ Fast and responsive UI
- 🎨 Modern glassmorphism design
- 📱 Mobile responsive
- 🔄 Loading state handling
- ✨ Smooth animations and hover effects

---

## 🛠 Tech Stack

- React
- JavaScript
- CSS3
- FreeAPI Random Jokes API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/random-joke-generator.git
```

Navigate to the project folder:

```bash
cd random-joke-generator
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
