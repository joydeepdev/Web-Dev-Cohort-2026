# 🍽 Meal Explorer

A modern React application that displays delicious meals using a public meals API with pagination support.  
Built with React Hooks and styled using a beautiful dark-themed responsive UI.

---

## ✨ Features

- 🔥 Fetch meals dynamically from API
- 📄 Pagination support
- 🎨 Modern dark UI
- 📱 Fully responsive layout
- ⚡ Fast and lightweight
- 🖼 Beautiful meal cards with hover effects
- ▶ Direct YouTube recipe links

---

## 🚀 Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- Fetch API

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/meal-explorer.git
```

Navigate into the project folder:

```bash
cd meal-explorer
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🌐 API Used

This project uses the public meals API:

```bash
https://api.freeapi.app/api/v1/public/meals
```

Example request:

```bash
https://api.freeapi.app/api/v1/public/meals?page=1&limit=12
```

## 🎯 Functionality

### Meals Display

- Displays 12 meals per page
- Shows:
  - Meal image
  - Meal name
  - Category
  - Area/Cuisine
  - YouTube recipe link

### Pagination

- Previous button
- Next button
- Current page indicator
