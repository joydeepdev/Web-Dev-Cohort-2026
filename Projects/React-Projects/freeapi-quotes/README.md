# ✨ Quote Generator App

A beautiful and minimal **Quote Generator** built with **React + Bun**.  
This app fetches inspirational quotes from a public API and displays them inside a modern glassmorphism UI.

---

## 🚀 Features

- 📜 Random quote generation
- ⚡ Built using **Bun**
- ⚛️ React Hooks (`useState`, `useEffect`)
- 🎨 Modern responsive UI
- 🔄 Fetch new quotes instantly
- 🌐 Uses FreeAPI public quotes API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/quote-generator.git
```

Move into the project folder:

```bash
cd quote-generator
```

Install dependencies using Bun:

```bash
bun install
```

Start the development server:

```bash
bun run dev
```

---

## 🌍 API Used

FreeAPI Quotes Endpoint:

https://api.freeapi.app/api/v1/public/quotes
